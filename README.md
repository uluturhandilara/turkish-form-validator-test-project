# Turkish Form Validator Test Project

React.js + TypeScript + Tailwind CSS ile oluşturulmuş bir proje.

## Kurulum

Bağımlılıkları yüklemek için:

```bash
npm install
```

## Geliştirme

Geliştirme sunucusunu başlatmak için:

```bash
npm run dev
```

Tarayıcıda [http://localhost:5173](http://localhost:5173) adresini açın.

## Build

Üretim build'i oluşturmak için:

```bash
npm run build
```

## Önizleme

Build edilmiş projeyi önizlemek için:

```bash
npm run preview
```
